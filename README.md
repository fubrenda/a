# A

A project to produce an easy to use authority server

## Contributing

1. Fork
2. Commit
3. Make sure tests pass
4. Create PR
5. Get code review

## Getting Started

This will install dependencies. You'll be able to run tests and
build the codebase afterwards.

```bash
make init
```

Run the tests

```bash
make test
```

To build

```bash
make build
```
